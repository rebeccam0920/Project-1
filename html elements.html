Introduction

Before considering style, you must learn the anatomy of the HTML5 tags:

  • An element is the tag that opens and closes around text that it describes.

  • The element closes when the same tag contains a backslash, or if it is a self-closing tag and is formatted as such.

  • An attribute is the text added to a tag to differentiate it from tags of the same variety.

  • The id attribute is used with the section div tags, while the class attribute is used with all other tags (when to use id and class is more complicated, but this will serve you well for now).

  • The href attribute is used with the a tag, while the src attribute is used with the img tag.

  • These are the basic concepts, and there is more to learn, but this will suffice for now. If you want to try to do more, make sure you have the basics down, then ask and I will work with you to expand on the basics.

Within paragraphs, there are a few important tags you will want to use. Sometimes, you will want to make certain text strong to make it stand out, like "when leaving the chamber, you must decontaminate!"; the strong tag will, by default, create bolded text; use it for text that is more important than other text. Likewise, you will want to emphasize certain phrases; the em tag, by default, will italicize that text and is used for text that deserves emphasis, like "I want that one, daddy!". If you want to reference the title of a publication, such as The Essentials of Technical Communication, use the cite tag; it is different from the emphasis tag because it denotes a book or publication title, not emphasized text.

Avoid junk code such as the span tag, which doesn't do anything except define styles for a block of text. Likewise, do not use non-breaking space characters like so   to space your text out (use CSS instead).

Links

The ability to link to other websites, documents, and objects is what that makes this document "hypertext," so lets look at hyperlinks and image tags:

Hyperlinks

Hyperlinks use the a tag and include the href attribute that points to the file or webpage you want text to link to. The text enclosed in the element is the "link" that you click on. For example, this is a link to Google. If you want to link to another page on your site, you can use a "relative link"; for example, this is a link to the index.html file in the example folder. The a tag is also used in your site navigation (the navigation div is just a list of links).

Images

You can add images directly into your page by using the img tag (note how it is self-closing). The src attribute points to the location of the image file. Use the alt attribute to provide information about the image for persons with low or no vision using screen readers.
